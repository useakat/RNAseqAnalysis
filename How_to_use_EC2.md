## ストレージの拡張方法
  1. AWSコンソールで、EC2インスタンスのストレージを増やす
  2. EC2にログインし、パーティションの状態を確認する
  3. ストレージの容量に合わせて、使用中のパーティションを拡張する
    拡張したいパーティションが /dev/nvme0n1p1 の場合
    ```bash
    sudo growpart /dev/nvme0n1 1
    ```
